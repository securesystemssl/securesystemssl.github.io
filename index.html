Quienes somos

Secure Systems es una empresa de instalacion de sistemas operativos. Nosotros hemos desarrollado un sistema operativo llamado "Secured OS" basado en Archlinux y ofrecemos el servicio de instalacion, configuracion y mantenimiento

Secured OS

Secured OS es un sistema operativo basado en Archlinux. Esta version de archlinux esta centrada en proveer diferentes servicios de forma segura. Por el momento estos son los servicios que ofrece Secured OS:
- Secured Web OS
- Secured FTP OS
- Secured Mail OS
- Secured Desktop OS
