Quienes somos

Secure Systems es una empresa de instalacion de sistemas operativos. Nosotros hemos desarrollado un sistema operativo llamado "Secured OS" basado en Archlinux y ofrecemos el servicio de instalacion, configuracion y mantenimiento

Secured OS

Secured OS es un sistema operativo basado en Archlinux. Esta version de archlinux esta centrada en proveer diferentes servicios de forma segura cara a ataques directos a la empresa. Por el momento estos son los servicios que ofrece Secured OS:
- Secured Web OS
- Secured FTP OS
- Secured Mail OS
- Secured Desktop OS

Secured Web OS

Sistema operativo servidor destinado a proveer servicio web de forma segura.

Secured Mail OS

Sistema operativo de servidor destinado a proveer sevicio de correo de forma segura

Secured FTP OS

Sistema operativo de servidor destinado a proveer servicio de transferencia de ficheros de forma segura

Secured Desktop OS

Sistema operativo segura de escritorio destinado a los equipos personales de la empresa
